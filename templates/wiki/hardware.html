Inventory

Inventory manager allows user to add, edit, and remove hardware. In addition, it can also generate reports with a list of all the hardware and what events it is assigned to.

 

Adding Hardware

To add hardware to the inventory:

  • Click on Tables on the left-side menu to expand, then click on Inventory

  • New page will open with fields for new hardware information

;

  • In the Serial Number field enter the serial number of hardware you wish to add (i.e. 1YQZ87). This field cannot be left blank.

  • Provide Description of hardware (i.e. Dell Laptop)

  • Configuration (if unknown it can be left blank)

  • Select the Type of hardware that is being added (i.e. laptop)

  • For Pool ID leave on Default Pool.

  • Click Save button to save the information and add the hardware to inventory.

 

Searching and Editing Hardware

Application features instant-search which allows for faster and instant searching of hardware items.
To find a specific piece of hardware, user can use the serial number. Alternatively, you can use description or hardware type as a search term to get a list of all the items that match that search term.

To find and edit a specific piece of equipment:

  • Click on Tables to expand the menu, then click on Inventory

  • Inventory manager page will open, with Search box near the top

  • To find the hardware we want to edit, enter the serial number

  • Instant search will immediately list all the items that have that serial number

 

  • Click on the serial number to edit information for that piece of hardware.

 

 

Generating Reports

To generate reports for hardware:

  • Click on Tables on the left side menu, then click Inventory.

  • A new page will open with list of hardware currently in the inventory.

The application offers 4 options for hardware reports: CSV, Excel, PDF, and Print. CSV, Excel, and PDF options allow the user to generate a file and save it in selected format. Print option opens a new page with a list of hardware that can be printed out.
