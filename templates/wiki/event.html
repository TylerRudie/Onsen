Events

To open the Events Calendar, click on the Event menu item.

The Event Calendar allows you to view information about ongoing and upcoming events as well as add new events.

 

 

Creating an Event 

  • Click on the Add Event button.

  • New page will open with fields for information related to the event

 

  • In the Title field enter the event's name (i.e. Lockheed-Martin Design Training).

  • For Start* enter or use the pop-up calendar to select start date (i.e. 2015-12.16).

  • For End* enter or use the pop-up calendar to select end date (i.e. 2015-12-18).

  • If the event will last entire day, check the All day option

  • Next Event allows you to select the following event the hardware will be sent to.

  • Pool field should be left on Default Pool.

  • Click on the "Shipping" tab. This allows you to enter related shipping information.

 

  • In the Date Shipped field enter or use the pop-up calendar to select 2015-12-16

  • Airbills To Event - airbill information for shipment sent to the event (i.e. airbill #: 222335888887989)

  • Airbills From Event - airbill information for shipment returning from the event (i.e. airbill #: 54987889987888)

  • CaseAssigned allows you to select a shipping case that has been assigned to this event. (In this case there aren't any cases assigned)

  • Click on the next tab labeled "Details". This page contains additional information about the event and requested hardware

  • In the Site field enter the name of client's site where the event will take place (i.e. Lockheed - Martin Collinsville)

  • Seat Revenue is the calculated revenue from laptop sent to the event. It can be left blank if unknown.

  • Projector Revenue is the calculated revenue from each projector sent to the event. It can be left blank if unknown.

  • Laptops Requested is the number of laptops the client has requested for the event.

  • If client requested projectors, then check the Projector Requested option

  • Config Assigned is the name of saved configuration for laptops. There isn't one for this event, so there's nothing to select.

  • Next, click on the "Contacts" tab. This tab allows you to select the shipping contact and the instructor for the event.

 

  • For Shipping Contact use the drop-down list to select Bruce Tanner.

  • Under Instructor Contact scroll down the list and click on Bruce Tanner to select him as the instructor for the event.

  • Finally, click on the Hardware Tab to select hardware which will be sent out to the event.

  • Under Hardware, select six Dell laptops

  • To remove hardware from the list, click on the item you wish to remove and it will be removed automatically.

  • Click on the Save button to save all the information you entered and create the event.

 

Editing an Event

To edit an event:

  • Click on the Events menu-item to go to the Events calendar

  • Select the event you wish to edit
