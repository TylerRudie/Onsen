Pools

Pools are used to divide hardware based on its availability. Initially, all hardware is added to the "Default Pool" which marks the hardware as available in the system so it can be selected during event creation. Additional pools can be created to group hardware that has been damaged, lost, retired, or set aside for internal use. Setting the hardware in Inventory with any pool other than Default will prevent that hardware from being checked out to an event.

On the left side menu, click on Tables then click on Pool to see list of pools currently in the system. From this page, you can add new pools or edit existing pools.

 

Creating a new Pool

To create a new pool:

  • Click on Tables then click on Pools.

  • Click on Add Pool button.

  • New page will open with fields for information related to the Pool.
