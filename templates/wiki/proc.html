Checkout/Check-in Hardware

To prevent the same hardware from being mistakenly selected for more than one event, it is important to checkout hardware out of the system. This reserves the hardware to the specific event and it will no longer appear on the list of hardware when creating a new event.
Likewise, once hardware comes back from an event, it should be checked in so it is available for the next event.

 

Check-out Hardware

To check out the selected hardware for the event we created:

  • Click on the Events link to go back to the Events calendar

  • Click on the event you wish to checkout hardware for (i.e. Lockheed-Martin Design Training)

  • Click on the Checkout Hardware button (above Basic Information tab)

  • On the following page we'll see a list of all the hardware we selected when we created the event

  • Select each box for each piece of hardware to be checked out and then click Checkout.

 

Check-In Hardware

To check-in hardware once it comes back from an event:

  • Click on the Event on the Calendar

  • Click on the Check-in Hardware button (above Shipping tab)

  • Next page will display a list of all the hardware that was checked out to that event. To check the hardware back into the system, select each hardware item and click on the Check in button

 

 

 

 

Printing Forms

Application can generate certain forms related to the event. These forms include Shipping Request Form and Packing Sheet. These forms can be printed or saved as PDF files.

 

Shipping Request Form


To generate a Shipping Request Form:

  • Click on an event for which you wish to generate the form

  • Click on the Shipping Request Form above the Contacts Tab

  • Application will automatically generate a PDF file and open it a new page

  • Generated form can now be printed out or saved to the computer as PDF

     

Packing Sheet

To generate a Packing Sheet:

  • Click on the Packing Sheet button

  • This will automatically generate a Packing Sheet PDF and open it in a new page
