Admin Panel

 

Admin panel provides additional functionalities and advanced features that are not available in the main interface. Some of these functions include the ability to delete contacts, events, hardware assignments, configurations, and pools. Additionally, the Admin Panel allows the user to create or delete groups and users that access the system. These functions are not available in the main interface for security reasons.


The Admin Panel can be accessed quickly from main interface by clicking on Extras and then Admin Panel.


It can also be accessed directly via following link: http://onsen.tech-res.net/admin.
To log in for the first time, you can use user admin and password system32

 

Users and Groups

The Admin Panel allows you to create additional users and user groups which will have access to the system. Groups make it easier to manage user permissions for multiple users so they do not have to be specified individually.

 

Creating Groups

To create a new group in Admin Panel:

  • Click on Groups link located under "Authentication and Authorization" section.

  • This will open up a new page and display a list of groups currently in the system

 

  • Click on Add Group

  • New page opens where you can enter the name for new group and select permissions which will be applied to users belonging to this group

 

 

Creating a new user


To create a new users that will have access to the system:

  • Click on Users link under "Authorization and Authentication" section.

  • New page will open and display a list of all the users currently in the system.

  • Click Create User button.

  • On the next page, enter username for the new user as well as password.

  • Click Save to save the information and continue.

  • New page will open which allows you to add the user to a group or assign special

  • On this page you can enter the user's first and last name as well as email address. When creating a new user it is important to make sure the option under Permissions section "Active" is selected, otherwise the user will be created as inactive user.

  • If you would like this user to be able to log in to the Admin Panel, you can select Staff Status option. If you would like to set this user up with all the permissions, check the Superuser Status option. NOTE: Giving a user Superuser permissions will allow them unlimited access to the system.

  • From the Available Groups you can select the group(s) you wish to add this user to. This will automatically give the user same permissions that are assigned to that group.

  • Additional or individual permissions can also be added under User Permissions.

  • Click Save button to save all the information and create the new user.
